# SaaS Landing Page

Modern, responsive SaaS landing page focused on clarity, conversion, and clean UI using HTML, TailwindCSS, Flowbite and JavaScript,

## Features

- **Hero Section** with value-driven headline and CTAs
- **Social Proof** section with trust indicators and statistics
- **Features Showcase** highlighting four main SaaS features
- **How It Works** three-step onboarding process
- **Pricing Section** with three-tier pricing model
- **SEO Optimized** with meta tags, Open Graph, and structured data
- **Responsive Design** with smooth scrolling navigation

## Tech Stack

- **HTML5** - Semantic markup with accessibility considerations
- **TailwindCSS** - Utility-first CSS framework for responsive design
- **Flowbite** - Modern TailwindCSS UI library
- **JavaScript (ES6+)** - Vanilla JavaScript for interactive features
- **Vite** - Modern build tool and development server

## Setup

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager

### Steps

```bash
git clone https://github.com/yourusername/project-name.git
cd project-name
npm install
```

## Build Process

The project uses Vite as the build tool. When you run `npm run build`, Vite will:

- Bundle and optimize all assets
- Minify HTML, CSS, and JavaScript
- Generate production-ready files in the `public` directory
- Optimize images and other static assets

## Usage

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
